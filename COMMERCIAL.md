# Commercial Services

fhir-codebridge is open source under the MIT license. The code is free.
Professional services are available separately for organizations that need
implementation support, managed hosting, or training.

## Implementation Consulting

For hospitals that need help deploying fhir-codebridge, integrating with
EHR systems, or customizing terminology mappings:

- **Scope:** Deployment, EHR integration (Epic/Cerner/Meditech), custom
  crosswalks, UMLS setup, RBAC configuration, audit log compliance review
- **Engagement:** Fixed-scope or time-and-materials
- **Contact:** See author profile on GitHub

## Managed Hosting

For organizations that want fhir-codebridge without managing infrastructure:

- **Scope:** Secure cloud or on-prem deployment, monitoring, backups,
  UMLS key management, version upgrades, SLA
- **Pricing:** Based on deployment size and support tier
- **Note:** PHI never leaves your network. Managed hosting means we manage
  the infrastructure, not the data.

## Training

For teams that need to understand terminology mapping, FHIR ConceptMap
operations, or confidence-based coding workflows:

- **Scope:** Workshops for coding teams, IT staff training, developer API
  training, compliance team briefings
- **Format:** On-site or remote, half-day to multi-day

## Open Source vs Commercial

| | Open Source | Commercial |
|---|---|---|
| Code | Free (MIT) | Same code |
| Support | GitHub issues (best effort) | SLA-backed |
| Deployment | DIY | Done for you |
| Customization | DIY | Expert implementation |
| Training | Read the docs | Workshops + hands-on |

The open source project is the product. Commercial services help
organizations adopt it faster. There is no "enterprise edition" with
features locked behind a paywall.
